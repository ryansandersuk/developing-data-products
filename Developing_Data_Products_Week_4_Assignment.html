Predicting fuel efficiency (miles per gallon)

The goal of this application is to see what predictors are best for predicting fuel efficiency. The dataset used is mtcars.

Evaluation is done simplisticly based on the R squared and adjusted R squared values, to show the variance explained by the model.

Not all possible predictors are considered.

Using lasso to select features build model

Using the lasso with cross validation to select features, can see that constraining a linear model (through lambda) results in a lower mean squared error.

x <- model.matrix(mpg ~ ., mtcars)[, -1]
y <- mtcars$mpg

model <- cv.glmnet(x, y, alpha = 1)
plot(model)

Using lasso to select features check coefficients

best_lambda <- model$lambda.min

grid <- 10^seq(10, -2, length = 100)

out <- glmnet(x, y, alpha = 1, lambda = grid)

coefficients <- predict(out, type = "coefficients", s = best_lambda)
